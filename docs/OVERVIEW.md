# money-log Project Overview

It's about planning, budgeting, and expense tracking using the Envelope Budgeting method (Zero-Based Budgeting).

## Core Features
*   **Zero-Based Budgeting:** Give every dollar a job.
*   **Transactions:** Log expenses, incomes, transfers, and bucket moves.
*   **Dashboard:** High-level view of Total Cash vs Total Budgeted.
*   **Action Bar:** Quick access to log transactions or view history.
*   **Settings:** Manage Accounts and Buckets (Create, Archive, Unarchive, Delete).
*   **Mobile-First Design:** Optimized for logging transactions on the go.
