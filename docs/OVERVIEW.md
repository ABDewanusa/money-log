# money-log Project Overview

It's about planning, budgeting, and expense tracking using the Envelope Budgeting method (Zero-Based Budgeting).

 ## Core Features
*   **Zero-Based Budgeting:** Give every dollar a job.
*   **Core Types:** Groups are mapped to a core philosophy type: **Need**, **Want**, **Savings**.
*   **Transactions:** Log expenses, incomes, transfers, and bucket moves. Fast modal entry via intercepting route.
*   **Transaction Filters:** Tabs on the Transactions page: **All | Income | Expense | Transfers**.
*   **Bottom Navigation:** Persistent tab bar for seamless switching between Dashboard, Transactions, Reports, Settings.
*   **Dashboard:** Budgets grouped by **Core Type** with account balances and a **Recent Activity** widget.
*   **Settings:** Manage Accounts, Groups, and Buckets (Create, Update, Archive/Unarchive, Delete). Drag-and-drop sorting.
*   **Optimistic UI:** Immediate visual updates on add/edit/archive actions, with server sync in background.
*   **Mobile-First Design:** Optimized for logging transactions on the go. Safe-area padding with fixed navigation.
